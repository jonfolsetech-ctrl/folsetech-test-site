# Folse Tech - Optimized Next.js Site

A minimal, high-performance Next.js website optimized for speed with zero unnecessary dependencies.

## Features

- **Minimal Dependencies**: Only Next.js, React, React-DOM, Tailwind CSS, and TypeScript
- **Performance Optimized**: Built-in SWC minification, image optimization, and production builds
- **Type Safe**: Full TypeScript support with strict mode enabled
- **SEO Ready**: Built-in JSON-LD structured data support
- **Responsive Design**: Mobile-first Tailwind CSS styling
- **No External Libraries**: Clean component architecture with pure React

## Quick Start

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Start production server:
```bash
npm start
```

## Performance Optimizations

✅ **No unnecessary dependencies** - Only essential packages  
✅ **SWC compilation** - Faster TypeScript compilation  
✅ **Production source maps disabled** - Smaller bundle size  
✅ **Strict TypeScript** - Catch errors at compile time  
✅ **Tailwind purging** - Only includes used CSS  
✅ **Image optimization** - WebP and AVIF formats  
✅ **X-Powered-By header removed** - Security & performance
